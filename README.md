# Research Paper Simplifier

Ask questions about research papers using AI. Upload a PDF or fetch from arXiv, then chat with your paper using Google's Gemini API.

## Features

- Upload research papers (PDF or arXiv)
- Ask questions in natural language
- Get answers with citations from the paper
- Use your own free Gemini API key

## Installation
```bash
# Clone and install
git clone https://github.com/BibekP01/research-paper-simplifier.git
cd research-paper-simplifier
pip install -r requirements.txt

# Add your Gemini API key to .env file
# Get free key at: https://makersuite.google.com/app/apikey
GEMINI_API_KEY=your_key_here
```
